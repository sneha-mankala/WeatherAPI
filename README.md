# 🌦️ Weather Application

## 📌 Description
This project is a simple and responsive Weather Application that allows users to check real-time weather information by entering a city name. It uses JavaScript Fetch API to retrieve data from the OpenWeatherMap API and displays it dynamically on the webpage.

## 🚀 Features
- Search weather by city name
- Displays temperature, weather condition, and details
- Real-time API data fetching
- Simple and clean UI
- Responsive design

## 🛠 Technologies Used
- HTML5
- CSS3
- JavaScript
- OpenWeatherMap API

## ▶️ How to Run
1. Open `index.html` in a browser
2. Enter a city name
3. Click on "Get Weather"
4. View weather details instantly

## 🔑 Note
This project requires an API key from OpenWeatherMap. Replace the API key in `script.js` before running.
